## 3. Installation

1. clone repository
```bash
git clone https://github.com/floppyist/tinycraft.git .
```

2. change directory & rename config.json.dist
```bash
cd tinycraft/etc/
mv config.json.dist config.json
```

3. change directory back to project root
```bash
cd ../../
```

4. run tinycraft
```bash
./main.py
```
